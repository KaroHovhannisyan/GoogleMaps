Project show you last events in google maps in your country
to run project you must 
1) clone project
2) npm i 
3)npm run server
4) npm start

you need connect MongoDb in your computer for DB 

i use 
1) React
2)Redux with middlewares (Redux-Saga)
3) Router v4
4) Eventbrite API
5) Google Maps 
6) Material UI

Backend

1) NodeJs
2)Express
3)PassportJs 
4)Json Web token 
5)Express validator
6) MongoDb with Mongoose
